# Chat App

## Usage
### Clone the git url
`git clone https://github.com/catmans1/chatapp.git`

 `cd chatapp`
### Install yarn
`yarn install`

### Run test
`yarn test`

### Check format code
`yarn lint`

### Run iOS
`yarn pod`

`yarn ios` -> for iPhone

`yarn ipad` -> for iPad

### Run Android
`yarn android`
